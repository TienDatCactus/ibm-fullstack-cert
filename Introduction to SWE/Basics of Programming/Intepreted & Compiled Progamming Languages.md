Intepreted programming: JS, Python, Lua
- Scripted hoặc scripting language
	- Vận hành qua trình biên dịch trên máy tính / browser
	- Trình biên dịch sẽ dịch các code sang ngôn ngữ của máy tính
	- Máy tính sẽ hoàn thành công việc còn lại
- Luôn cần 1 trình biên dịch (interpreter) để dịch source code
- Các trình biên dịch này (hay gọi là translators) được dựng ngay trong browser hoặc yêu cầu 1 chương trình trên máy để chạy

Compiled Programming: C, C++, C#, Java
- Chương trình chạy trên máy tính
- Được đóng gói hoặc biên dịch thành 1 tệp có thể thực thi (executable)
- Thường là các chương trình lớn
- Giúp giải quyết các vấn đề phức tạp, như là BIÊN DỊCH source code
