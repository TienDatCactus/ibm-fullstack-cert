Các cấp độ của ngôn ngữ lập trình
- Ngôn ngữ lập trình cấp cao: SQL, Pascal, Python
	- Phức tạp hơn
	- Sử dụng tiếng anh phổ thông 
- NNLT cấp thấp: ARM, MIPS, X86
	- Dùng các kí tự đơn giản để đại diện cho ngôn ngữ máy

Query languages: SQL, AQL, CQL
- Là 1 yêu cầu lấy thông tin từ database
- DB sẽ tìm kiếm các bảng của nó cho thông tin đã được yêu cầu -> Trả về kết quả
- Quan trọng là cả ứng dụng tạo query đó và db xử lý query đó phải dùng chung 1 ngôn ngữ
- Viết query có nghĩa là viết những hướng dẫn có sẵn và dễ hiểu để tạo thành 1 yêu cầu hoàn chỉnh trong db
- Đạt được thông qua ngôn ngữ lập trình (query language/db query language)
- NoSQL (Not only SQL)
	- Khác biệt về cấu trúc dữ liệu (so với SQL)
	- SQL:
		- Có tính chất quan hệ
		- Sử dụng các schema (lược đồ) có cấu trúc, được định sẵn
	- NoSQL:
		- Không có tính chất quan hệ
		- Sử dụng schema động cho loại dữ liệu phi cấu trúc
- Yêu cầu dữ liệu, CRUD trong DB

Assembly languages
- Ít phức tạp hơn ngôn ngữ truy vấn, nnlt có cấu trúc và nnlt hướng đối tượng
- Dùng các kí tự đơn giản để đại diện 0 và 1
- Có liên kết chặt chẽ với kiến trúc CPU - mỗi loại CPU sẽ có loại assembly language riêng
- Được dịch sử dụng assembler thay vì compiler hay interpreter
- 1 statement <-> 1 instruction, ngược lại với nnlt cấp cao
