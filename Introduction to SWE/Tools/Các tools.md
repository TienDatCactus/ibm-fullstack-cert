CI/CD:
- Continuous Integration và Continuous Delivery/Deployment
- Cho phép các dev đưa ra các thay đổi thường xuyên 1 cách đáng tin cậy
- Thực hiện qua máy chủ build tự động
- CI:
	- Đảm bảo các thành phần trong code hoạt động mượt mà
	- Cho phép tích hợp code mới thường xuyên tùy thuộc vào tốc độ thay đổi của dự án
	- Tự động build và chạy code
- CD: bắt đầu khi CI kết thúc, deploy các thay đổi lên môi trường staging/testing
